3.评论列表

GET/POST:http://twapi.live.hoge.cn/index.php?m=Apituwenol&c=thread&a=show_comment

请求参数

参数 类型 格式 必填 说明
offset integer N 分页
count integer N 分页
topic_id integer N 话题id
thread_id integer N 报道id
status integer N 审核状态
create_time integer N 创建时间（时间戳）
user_id integer N 用户id

请求示例

http://twapi.live.hoge.cn/index.php?m=Apituwenol&c=thread&a=show_comment&topic_id=1098

返回

[]
